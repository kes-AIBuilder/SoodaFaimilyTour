# **App Name**: 2026 (언)수다(혜) 설특집 경주 패밀리 아케이드 웹앱

## Core Features:

- D-Day Timer: 실시간으로 2026년 2월 15일까지 남은 시간을 표시합니다.
- Interactive Tab System: 출연진, 큐시트, 게임/도구, 체크/맛집 4개 섹션으로 구성된 탭 시스템입니다.
- 6인 캐릭터 도감: 멤버별 아이콘, 태그, 하트 인터랙션이 있는 캐릭터 도감입니다.
- 타임라인 (큐시트): 경주 여행의 구체적인 일정을 시각적으로 보여주는 큐시트입니다.
- 랜덤 룰렛: 다음 게임을 정해주는 동적 룰렛입니다.
- 멀티 타이머: 인물 퀴즈(3초), 몸으로 말해요(60초) 등 예능 최적화 프리셋 타이머입니다.
- 랜덤 제시어 생성기: 인물, 노래, 몸말 등 카테고리별 데이터베이스 기반 무작위 제시어 출력 기능입니다.
- 체크리스트: 완료 시 폭죽 효과가 터지는 피드백 시스템을 갖춘 체크리스트입니다.
- Theme Song Generation: Leverage a tool that will find out what type of vacation it is. Based on what the vacation consists of, pick an appropriate theme song using AI. Play theme song during vacation.

## Style Guidelines:

- 고채도의 옐로우/핑크 컬러 팔레트를 사용합니다.
- Black Han Sans 등 굵고 강렬한 자막 폰트 느낌의 타이포그래피를 적용합니다.
- Lucide-react, Font-awesome 아이콘을 활용합니다.
- 모바일에서 한 손으로 조작 가능한 큰 버튼과 하단 네비게이션을 배치합니다.
- 클릭 시 폭죽, 하트 애니메이션 등 시각적 즐거움을 주는 고퀄리티 마감 처리를 합니다.
